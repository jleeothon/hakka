# Date and time expressions

Units of time that need classifiers:

| Unit            | Classifier |
| --------------- | ---------- |
| 年 (ñěn)        | -          |
| 月 (ñēt)        | 只 (zǎk)   |
| 日 (ñǐt)        | -          |
| 钟头 (zúng těu) | 只 (zǎk)   |
| 分钟 (fún zúng) | -          |
| 秒 (miāu)       | -          |

Time of the day:

| Example | Hanzi      | Reading            |
| ------- | ---------- | ------------------ |
| 8:30    | 八点半     | bǎk diâm bān       |
| 8:25    | 八点五只支 | bǎt diâm n̂g zǎk zī̵ |
